# Bank-App (C# Console Application)
A minature banking system built in C# as part of my volunteer developer work for Web Surfing Studios. This application simulates banking operations through a clean, menu-driven console interface.

## Features
### Account Management 
- Open new accounts
- Close existing accounts
- View account balances

### Transactions 
- Deposit funds
- Withdraw funds
- Validation for insufficient funds
- Clear error handling and user feedback

### Data Handling 
- In-memory account storage
- Unique account tracking
- Clean separation of concerns using classes (Bank, Account, etc.)

### User Experience 
- Menu-driven interface
- Input validation
- Helpful error messages
- Consistent command logic

--- 

## How to Run the Application
1. Download Visual Studio 2022 and Git
2. Create a folder in your C drive
3. Open Git Bash in the folder (Right click the folder and select Open Git Bash Here)
4. Type command git init
5. Clone the repository by typing the command: git clone https://github.com/Klenburg/Bank-App.git
6. Open the project in Visual Studio 2022 (right click the entire BankApp folder and select open in Visual Studio)
7. Go to the Solution Explorer and select BankApp.sln
8. Run the console application by hitting the green play button
