# Bank-App (C# Console Application)
A minature banking system built in C# as part of my volunteer developer work for Web Surfing Studios. This application simulates banking operations through a clean, menu-driven console interface.

## Features
### Account Management 
- Open new accounts
- Close existing accounts
- View account balances

### Transactions 
- Deposit funds
- Withdraw funds
- Validation for insufficient funds
- Clear error handling and user feedback

### Data Handling 
- In-memory account storage
- Unique account tracking
- Clean separation of concerns using classes (`Bank`, `Account`, etc.)

### User Experience 
- Menu-driven interface
- Input validation
- Helpful error messages
- Consistent command logic

--- 

## How to Run the Application
1. Clone the repository: https://github.com/Klenburg/Bank-App.git
2. Open the Project in **Visual Studio 2022**
3. Build the solution
4. Run the console application
